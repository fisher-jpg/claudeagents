# Claude Agents — John Fisher

Backup of custom Claude skills, memory edits, and AI workflow assets.

## Contents

### `/skills/`
Custom Claude skills that trigger automatically based on conversation context.

- **`demo-plan/`** — Converts free-form sales prep notes into a structured one-pager demo plan (Markdown output)
- **`demo-prep-summary/`** — Similar to demo-plan with additional sections: Key Risks/Watch-Outs, question rationales, stakeholder-aware demo flow, and an "Info to Gather" checklist for sparse notes

### `/memory-edits/`
Persistent instructions that shape Claude's behavior across conversations (e.g., role context, workspace guardrails).

## About
These assets support my work as a Solutions Consultant at Gem, where I use Claude to streamline demo prep, prospect research, and sales enablement workflows.
